# tuya-iotos-embeded-demo-zigbee-sdk-development-course
Tuya IoTos Embeded Demo Zigbee SDK Development Course


## demo说明

本例程是以一路开关为例演示如何使用TuyaOS SDK开发Zigbee开关产品各种功能；

配网方式：复位按键或长按配网按键 5s 以上；

APP：使用涂鸦智能 APP
